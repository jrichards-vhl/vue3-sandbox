# vue3-sandbox

## Project setup
```
npm install
```

### Compiles and hot-reloads for development

The project is set to run on 127.0.0.1:3003 by default.  Ensure that this port
isn't being used by another service before running.

```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
